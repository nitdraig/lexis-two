# Release v1.1.1

Portable rules, skills, and slash commands for AI agents — first recommended release for adoption.

## Install

```bash
npm install -D @draig/lexis-two
# or one-shot setup in your project:
npx @draig/lexis-two install --host cursor,opencode,agents --scope project --yes
```

OpenCode users: the **plugin** injects rules; **slash commands** need the install step above (or global: `--scope global`).

## Highlights

- **`npx @draig/lexis-two install`** — Cursor, Windsurf, Cline, Kiro, OpenCode, AGENTS.md, Copilot repo rules
- **Unified `/lexis`** — `review`, `audit`, `debt`, `plan`, `security`, `help`, mode switches (`lite` / `full` / `ultra` / `off`)
- **Nine before/after examples** — Next.js, Express, FastAPI (`examples/`)
- **`CONTRIBUTING.md`** — how to add hosts, skills, and commands without duplicating logic
- **OpenCode fixes** — commands in `.opencode/commands/`; Windows global path `~/.config/opencode`
