# Release v1.2.2

Cursor Agent Skills — same portable `skills/` as OpenCode, copied into `.cursor/skills/`.

## Install / upgrade

```bash
npm install -D @draig/lexis-two@1.2.2
# project rules + skills:
npx @draig/lexis-two install --host cursor --scope project --yes
# all projects (global skills):
npx @draig/lexis-two install --host cursor --scope global --yes
```

Reload the Cursor window after install.

## What lands in Cursor

| Source | Destination |
